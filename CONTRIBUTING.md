# Contributing to CanIReach

Welcome to the CanIReach project! We appreciate your help in building a better network diagnostics console.

Please refer to the comprehensive **[Contributing Guidelines](docs/contributing.md)** inside the `docs` folder for:
* Bug reporting templates
* Local code verification standards (Clippy, formatting, tests)
* Guidance on suggesting and adding default target endpoints to the built-in database

Thank you for your support!
